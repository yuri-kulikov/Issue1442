# Issue1442
Simple app to test CodePush.sync on componentDidMount

### How to run:
1. `git clone https://github.com/yuri-kulikov/Issue1442.git`
2. `cd Issue1442`
3. `npm install`
4. Run: 
   - `react-native run-android` (then close bundler)
   - or run from **Android studio**
